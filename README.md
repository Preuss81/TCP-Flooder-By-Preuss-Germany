# Preuss TCP-Flooder

Proxy-Prüfung und TCP-Flood-Simulation mit Endlosschleife, Registry-Speicherung und Exportfunktion. Entwickelt für transparente Tests im lokalen Netzwerk.

## Version

**TCP-Flooder.exe Version 1.1.133**  
**Herausgeber:** Preuss Germany

## Funktionen

- Proxy Graber: Automatische und manuelle Suche nach öffentlichen Proxies
- Proxy Checker: Prüft die Funktion und Erreichbarkeit jedes Proxyservers
- Port Finder: Ermittelt offene Ports auf Zielservern zur Analyse
- TCP-Flooder: Simuliert gezielte TCP-Anfragen zur Proxyprüfung
- Endloser Durchlauf der Proxy-Liste mit automatischem Neustart
- Speicherung guter Proxies in der Registry
